# Fiitsio - Smart Gym Management Platform

A production-ready landing page for Fiitsio, the all-in-one gym management SaaS platform.

## Overview

Fiitsio helps gym owners automate operations, engage members, and grow revenue with features including class booking, member progress tracking, integrated payments, and push notifications.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox, animations
- **Vanilla JS** - No dependencies
- **Google Fonts** - Inter typeface

## Features

- Fully responsive mobile-first design
- Sticky navigation with hamburger menu
- CSS phone mockups (no images required)
- FAQ accordion
- Smooth scroll navigation
- Scroll-triggered fade-in animations via IntersectionObserver
- App Store / Google Play download buttons

## Development

Open `index.html` in a browser. No build step required.

```bash
# Quick local server
python3 -m http.server 8000
```

## Deployment

Static site — deploy to any hosting provider (Netlify, Vercel, GitHub Pages, S3, etc.).
